Installation instructions:
1. Purchase and install Portal from Steam.
Assuming C:\Program Files (x86)\Steam\steamapps\common\Portal\ is the game folder:
2. If you've installed a previous version of the mod,
first remove portal\custom\portl under the game folder and restore the backup of portal\resource\portal_english.txt if
needed. If the file doesn't exist and there's no backup verify the game files in Steam.
3. Extract the contents of the zip into
C:\Program Files (x86)\Steam\steamapps\common\Portal\portal

In order to uninstall, Remove the portal\custom\portl folder
