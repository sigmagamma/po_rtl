Installation instructions:

1. It is assumed you're running Windows 10/11 with Hebrew support.
2. Purchase and install Portal.
Assuming C:\Program Files (x86)\Steam\steamapps\common\Portal\ is the game folder:
3. Run the exe file and follow the instructions to install the patch.
4. *** The following changes the display font for your entire system. *** Run apply_fonts_portal_hebrew.reg.
Restart your PC.

In order to uninstall:
1. Run the exe file and follow the instructions to uninstall
2. Remember to run remove_fonts_portal.reg file. You may have to restart Windows.
