Installation instructions:

1. It is assumed you're running Windows 10/11 with Hebrew support.
2. Get Portal RTX from Steam
Assuming C:\Program Files (x86)\Steam\steamapps\common\PortalRTX\ is the game folder:
3. Extract the contents of the zip into the game folder
4. *** The following changes the display font for your entire system. *** Run apply_fonts_portal_hebrew.reg.
Restart your PC.
5. Within the game in the options menu, make sure captioning is turned on and language is set to English.

In order to uninstall:
1. Remove the portal_rtx\custom\portl folder
2. Remember to run remove_fonts_portal.reg file. You may have to restart Windows.
