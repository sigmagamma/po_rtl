This is an alpha version. By following these instructions you acknowledge you are responsible for doing so.
Installation instructions:

1. Run the exe file and follow the instructions to install the patch. 
2. It is assumed you're running Windows 10 with Hebrew support, meaning your default system Hebrew font is Tahoma.
3. *** The following changes the display font for your entire system. *** Run applyfonts.reg to change the default font to Miriam Fixed. Restart your PC.
4. When you start the game, set the game resolution to 1366 * 768 (higher resolutions might work as well, but the subtitles aren't as readable)


In order to uninstall:
1. Run the exe file and follow the instructions to uninstall
2. Remember to run removefonts.reg file and restart Windows.
