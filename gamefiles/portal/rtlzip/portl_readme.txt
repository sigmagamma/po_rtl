This is a beta version. By following these instructions you acknowledge you are responsible for doing so.
Installation instructions:

1. It is assumed you're running Windows 10/11 with Hebrew support.
2. Purchase and install Portal.
Assuming C:\Program Files (x86)\Steam\steamapps\common\Portal\ is the game folder:
3. If you've installed a previous version of the mod,
first remove portal\custom\portl under the game folder and restore the backup of portal\resource\portal_english.txt if
needed. If the file doesn't exist and there's no backup verify the game files in Steam.
4. Extract the contents of the zip into the game folder
5. *** The following changes the display font for your entire system. *** Run apply_fonts_portal_hebrew.reg.
Restart your PC.


In order to uninstall:
1. Remove the custom\portl folder
2. Remember to run remove_fonts_portal.reg file. You may have to restart Windows.
