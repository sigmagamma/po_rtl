מסמך זה מכיל הוראות התקנה והסרה לטלאי העברית של פורטל 2 בלינוקס.
הטלאי הוא תוצר של עבודה לא-מסחרית ואין עליו אחריות - מעקב אחר הוראות אלה מהווה לקיחת אחריות על כל השלכה.

הוראות התקנה

התקינו את פורטל 2 בסטים.
הריצו את המשחק כדי לוודא שיש לכם את הגרסא העדכנית ביותר.
במידה ויש לכם מודים מותקנים בתיקיה portal2_dlc3 שתחת התיקיה הראשית של פורטל 2, מומלץ לגבות ולהסיר אותם.
פרסו את קובץ הזיפ לתיקייה הראשית של פורטל 2.
לאחר הדריסה תראו בתיקייה הראשית תיקייה בשם portal2_dlc3, לצד תיקיות כמו portal2_dlc2 שהיו קיימות לפני כן.

הסרת ההתקנה

כדי להסיר, הסירו את התיקייה portal2_dlc3 מהתיקייה של פורטל 2.
לאחר מכן כנסו להגדרות המשחק בסטים (גלגל השיניים, properties).
לחצו על Installed Files ובחרו ב Verify Game Files.

הפעולה Verify Game Files מוחקת קבצים של הגרסא העברית.
במידה ואתם חייבים לבצע אותה אבל תרצו להמשיך לשחק בעברית,
יהיה עליכם לבצע את ההתקנה מחדש.
